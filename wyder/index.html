WYDER

A Windows Video Downloader that doesn't suck.

GitHub

Wyder is an intuitive and powerful video downloader for Windows with no ads and no scams that allows you to download videos from various platforms, including YouTube, Twitch, TikTok, and many more, for free. It supports multiple formats and resolutions, ensuring you get the best quality for your needs.

⚠️ IMPORTANT

Video metadata will not be altered, which is why videos might not appear at the top of the Download folder if sorted by date. If you don't see your downloaded videos, try using the search function in File Explorer or sort by name. A fix for this issue is in the works.
