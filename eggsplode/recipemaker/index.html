IQBit >> Eggsplode >> Recipe Maker

Use this site to create recipes for the Eggsplode game!

1. Choose the cards

2. Copy the code

3. Enter the code in Eggsplode

  1. In Discord, send the /start command if you haven't already.
  2. Under Recipe, click on Edit.
  3. Replace the code in the Recipe JSON text field with the code you just copied. You can do this quickly by pressing Ctrl+A, then Ctrl+V.
  4. Click Submit to update the recipe.

4. Done! Now you can start the game.
