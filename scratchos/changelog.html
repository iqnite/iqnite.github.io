What's new in v3.5.3 (20. 10. 2024) Latest

Developmental changes:
  • Apps from the store are now on the new codebase that was inroduced in v3.5.0 and easier to develop

What's new in v3.5.2 (22. 9. 2024)

Bug fixes:
  • Meower Chat: Clicking outside the New Message button should no longer trigger the message input box

What's new in v3.5.1 (31. 8. 2024)

Changes and improvements:
  • The Load button now shows a badge indicating that it's loading
Bug fixes:
  • Paint: Opening the app should no longer draw a dot at the mouse position

What's new in v3.5.0 (22. 6. 2024)

New features:
  • Music: New "Loop" button allows to play songs repeatedly
Changes & improvements:
  • New home screen animation
  • Space Game: Background music will not play if there's a song already playing in the Music app
  • The WiFi button now also displays the username
Bug fixes:
  • Improved overall app performance
  • Gobo Assistant: Fixed the speech bubble not hiding when clicking outside of it
  • Calculator: Fixed the number fields not being hidden when going to the home screen
  • Calculator: Fixed the number fields being cleared every time the app is launched
Removed features:
  • Removed drag&drop support for app icons
    Note: This removal is only temporary. It is planned to be re-added in the next release.

What's new in v3.4.2 (17. 9. 2023)

Changes & improvements:
  • Improved positions and sizes of app icons
  • Increased contrast of some items
Bug fixes:
  • Fixed a bug that didn't allow leaving Meower Chat on some devices
  • Fixed a bug that didn't uninstall apps on hard reset

What's new in v3.4.1 (17. 9. 2023)

Changes & improvements:
  • App Store: Background now matches the system's color settings
Bug fixes:
  • Fixed a bug that reset alarms on startup
  • Fixed a bug that didn't allow to click the notifications button with "auto-hide taskbar" enabled

What's new in v3.4.0 (13. 9. 2023)

New features:
  • New app: Speed tester (allows to run CPS, FPS, FLOPS, and Cloud speed tests)
  • Data is no longer deleted at every startup
Changes & improvements:
  • The panel now has floating edges
  • Translator: If no language is specified, it will use your Scratch language
Bug fixes:
  • Enabling turbo mode should no longer make the screen glitch
  • Many other bug fixes regarding the system kernel
Removed features:
  • Legacy speed test (use the new Speedtest app)
  • Command line (was dangerous)

What's new in v3.3.1 (16. 2. 2023)

Changes & improvements:
  • Light/dark mode doesn't affect the home screen anymore
  • Added experimental support for forkphorus when pressing [x] on the error page
Bug fixes:
  • Fixed volume/internet displays not hiding when dismissing them
  • Music: Fixed some songs not playing
Removed features:
  • Paint: Removed Brightness slider because of issues with the Save/Load system (the feature is still available with Dev mode)

What's new in v3.3.0 (8. 2. 2023)

New features:
  • Re-added microphone volume display to Volume button in taskbar
  • Music: Added ability to play/pause sounds
  • Music: Added new song
  • Paint: Added ability to change brightness and saturation
Changes and improvements:
  • Paint: Sliders now hide when clicking outside
  • Now a warning is shown when trying to save a Notepad file containing placeholder strings
  • Login is no longer required to use Dev mode
Bug fixes:
  • Paint: Fixed some painting lines not saving
Known issues:
  • Music pause only works properly on TurboWarp with no-limits enabled and on the web player

What's new in v3.2.5 (15. 12. 2022)

Changes & improvements:
  • Meower: Chat notifications are now shown in taskbar
Bug fixes:
  • SpaceGame: fixed background showing only in Color theme
  • Fixed Network icon shwoing connection even when you're not logged in
Known issues:
  • Notepad + Translate: read aloud doesn't support more than 128 characters.
  • Gobo: voice output ignores actual volume
  • Gobo: when Auto-hide taskbar is enabled, the Gobo icon is hidden even if the Gobo app is opened
  • Calculator: exponents with point return wrong results
